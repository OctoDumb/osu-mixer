# osu! mixer

A simple player for osu! songs

## Requirements

- **Node.JS v10+**

## How To Run

### 1. App

*Coming some time soon...*

### 2. Code

1. **Clone git repository**
`git clone https://github.com/OctoDumb/osu-mixer.git`

2. **Install dependencies**
```shell
npm i sass -g
npm i
```

3. **Compile styles**
`npm run styles`

4. **Run**
`npm start`
